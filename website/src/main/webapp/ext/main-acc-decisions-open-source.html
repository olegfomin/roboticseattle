Open Source and Free projects used in the S.P.A.R. development

  • Raspbian - the Linux derivative used for the 'Raspberry PI' by default
  • Amazon Linux AMI - the default OS on EC2
  • Amazon's EC2 - this is the unbeatable offer. Those guys rule!
  • Eclipse Jetty - a very light-weight Java based Http Server and Servlet Engine
  • Oracle's Java
  • Google's Pico text-to-speech engine
  • Google's gson - very useful Java library to serialize/deserialize JSON
  • Eclipse - Java IDE for the computes with substantial RAM
  • BlueJ - small foot-print Java IDE for the less-powerful boards
  • librxtx-java - enables serial port communication
  • Arduino IDE for C++ allows the development and uploading
  • JQuery - the state of art java-script library

Though Webstorm by Jetbrains is not free but it is still $80 very well spent if you are serious about web-development.
